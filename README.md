### [VisualProof](https://github.com/Aston1690/visualproof)

Evidence-first visual QA for websites: real Chromium screenshots, deterministic UI checks, and shareable self-contained HTML + JSON reports.

```bash
npx visualproof audit https://example.com --output visualproof-report
```

- Audits desktop and mobile layouts in Chromium
- Detects overflow, missing metadata, heading problems, inaccessible images, small touch targets, console errors, and oversized typography
- Generates portable visual evidence instead of subjective “AI taste scores”
- Built for designers, frontend developers, agencies, QA teams, and coding agents

## Current focus

I am concentrating on one substantial open-source product rather than publishing more thin repositories.

For VisualProof, the priorities are:

- More deterministic visual and accessibility checks
- CI score thresholds
- Before/after comparison reports
- Contributions based on real frontend QA problems
